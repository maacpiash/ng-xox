## Install

```sh
hub clone maacpiash/ng-xox

cd ng-xox
```

## Usage

First, install the dependencies via NPM.

```sh
npm i
```

Then, run this command to use the application.

```sh
ng serve
```

## Run tests

```sh
ng test

ng e2e
```
